# 다차원 배열
## 2차원 배열의 선언과 인덱스
```java
int[][] score;
int score[][];
int[] score[];

int[][] score = new int[4][3];
```
## 2차원 배열 초기화
```java
int[][] arr = new int[][]{ {1, 2, 3}, {4, 5, 6} };
int[][] arr = { {1, 2, 3}, {4, 5, 6} };             // new int[][] 생략 가능
int[][] arr = {
                    {1, 2, 3},
                    {4, 5, 6}
                };
```

## 가변 배열
```java
int[][] score = new int[5][];
score[0] = new int[4];
score[1] = new int[3];
score[2] = new int[2];
score[3] = new int[2];
score[4] = new int[3];

int[][] score = {
                    {100, 100, 100, 120},
                    {20, 20, 20},
                    {30, 30},
                    {40, 40},
                    {50, 50, 50}
                };
```
- 다차원 배열을 생성할 때 전체 배열 차수 중 마지막 차수의 길이를 지정하지 않고, 추후 각기 다른 길이의 배열을 생성하여 유동적인 가변 배열을 구성할 수 있음
