# 객체 지향 언어

- 기존의 프로그래밍 언어에 몇 가지 새로운 규칙을 추가한 보다 발전된 형태
- 코드 간에 서로 관계를 맺어 줌으로써 보다 유기적으로 프로그램을 구성하는 것이 가능해짐
- 객체 지향 언어의 주요 특징은 다음과 같음
    1. 코드의 재사용성이 높음
        - 새로운 코드를 작성할 때 기존의 코드를 이용하여 쉽게 작성할 수 있음
    2. 코드의 관리가 용이
        - 코드 간의 관계를 이용해서 적은 노력으로 쉽게 코드를 변경할 수 있음
    3. 신뢰성이 높은 개발을 가능하게 함
        - 제어자와 메서드를 이용해서 데이터를 보호하고 올바른 값을 유지하도록 함
        - 코드의 중복을 제거하여 코드의 불일치로 인한 오동작을 방지할 수 있음
- 객체 지향 언어의 가장 큰 장점은 **코드의 재사용성이 높고 변경에 유리**하다는 것
- 이러한 객체 지향 언어의 장점은 프로그램의 개발과 유지 보수에 드는 시간과 비용을 획기적으로 개선
