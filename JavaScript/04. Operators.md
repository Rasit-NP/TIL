# Operators
### 연산자 종류
- 할당 연산자
- 증가 & 감소 연산자
- 비교 연산자
- 동등 연산자
- 일치 연산자
- 논리 연산자

### 할당 연산자
```javascript
let a = 10
a += 10
a -= 3
a *= 10
a %= 7
```
- 오른쪽에 있는 피연산자의 평가 결과를 왼쪽 피연산자에 할당하는 연산자
- 단축 연산자 지원

### 증가 & 감소 연산자
```javascript
let x = 3
const y = x++
console.log(x, y)   // 4 3

let a = 3
const b = ++a
console.log(a, b)   // 4 4
```
- 증가 연산자(`++`)
    - 피연산자를 증가시키고 연산자의 위치에 따라 증가하기 전이나 후의 값을 반환
- 감소 연산자(`--`)
    - 피연산자를 감소시키고 연산자의 위치에 따라 감소하기 전이나 후의 값을 반환
- 코드의 가독성을 위해 `a += 1`, `a -= 1`과 같이 더 명시적인 표현을 권장

### 비교 연산자
```javascript
3 > 2       // true
3 < 2       // false

'A' < 'B'   // true
'Z' < 'a'   // true
'가' < '나' // true
```
- 피연산자들을 비교하고 결과 값을 boolean으로 반환하는 연산자

### 동등 연산자
```javascript
console.log(1 == 1)              // true
console.log('hello' == 'hello')  // true
console.log('1' == 1)            // true
console.log(0 == false)          // true
```
- 두 피연산자가 같은 값으로 평가되는지 비교한 후 boolean 값을 반환
- 암묵적 타입 변환을 통해 타입을 일치시킨 후 같은 값인지 비교
- 두 피연산자 모두 객체일 경우 메모리의 같은 객체를 바라보는지 판별

### 일치 연산자
```javascript
console.log(1 === 1)              // true
console.log('hello' === 'hello')  // true
console.log('1' === 1)            // true
console.log(0 === false)          // false
```
- 두 피연산자의 값과 타입이 모두 같은 경우 `true`를 반환
- 같은 객체를 가리키거나, 같은 타입이면서 같은 값인지를 비교
- 엄격한 비교가 이뤄지며 암묵적 타입 변환이 발생하지 않음
- 특별한 경우를 제외하고는, 예측하지 못한 결과를 방지하기 위해 **일치 연산자(`===`) 사용을 권장**

### 논리 연산자
- and 연산
    - `&&`
- or 연산
    - `||`
- not 연산
    - `!`
- 단축 평가 지원
